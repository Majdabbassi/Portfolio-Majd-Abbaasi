Deployment Lab

Self-driven infrastructure training initiative.

To deeply understand production behavior, I repeatedly deployed a simplified application using multiple strategies — refining server setup, configuration logic, and release flows.

Hands-on experimentation included:
▸ Nginx reverse proxy (direct deployment)
▸ Apache HTTP Server configuration
▸ Docker containerization
▸ Docker Compose multi-service orchestration
▸ Environment variable management
▸ Service restart & failure recovery testing

Not a product — an engineering lab for mastering deployment from the ground up.
